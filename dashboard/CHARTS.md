## Architecture Notes

- Each chart group is a `<section class="chart-panel">` in `index.html`
- Data is pre-generated by `tools/export-dashboard-data.js` into `dashboard/data/*.json`
- New charts require: (1) add computation to export script, (2) add chart div to HTML, (3) add render function to `charts.js`
- Multi-planet overlay: all selected planets render on the same chart with planet-colored traces
- Earth-only panels auto-hide when Earth is not selected
